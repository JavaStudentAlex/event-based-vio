**Milestone:** M002

## UAT Type

- UAT mode: runtime-executable
- Why this mode is sufficient: The slice modifies internal numerical coordinate transformations, and synthetic tests with exact expected transforms correctly prove it.

## Preconditions

The python environment is ready and pytest is installed (`uv run pytest`).

## Smoke Test

Verify all tests under `tests/baselines/test_event_imu_backend.py` pass.

## Test Cases

### 1. Extrinsics Apply Properly 

1. Create an `EventImuBackend` processing an `MvsecSequence` with a known `T_imu_cam` transform in the dataset calibration.
2. Run backend and inspect positions. 
3. **Expected:** Generated trajectory diverges from an identically initialized run without extrinsics, confirming the rotation is evaluated.

### 2. Missing Extrinsics Fallback

1. Execute the backend on a dataset containing no extrinsics or degenerate `T_imu_cam`.
2. Inspect `EventImuBackend.diagnostics`.
3. **Expected:** `extrinsics_source` reports `identity_fallback`.

## Edge Cases

### Degenerate matrix

1. Apply an invalid `T_imu_cam` matrix (all NaNs or determinant far from 1).
2. **Expected:** System falls back gracefully to `identity_fallback` without crashing.

## Failure Signals

- Any crash or linear algebra error emitted from scipy `Rotation.from_matrix()`.
- Degradation or test failure in `tests/baselines/test_event_imu_backend.py`.

## Not Proven By This UAT

- We do not prove translational accuracy using lever-arm parameters (deferred).
- We do not formally prove improved scale/ATE on real sequences until S03 completes.
